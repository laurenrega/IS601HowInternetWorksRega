DNS

What is DNS?

DNS stands for domain name system. The DNS Associates domain names with the corresponding IP addresses. Your computer uses DNS to look up domain names and look up the IP address which is used to connect the computer to the requested domain.

DNS servers are connected in a distributed hierarchy. It is split up into decent zones.

DNS was created to be an open public communication protocol. Unfortunately, this exposes the system to cyber attacks. One particular cyber attack that DNS is vulnerable to is called DNS spoofing. This occurs when a hacker goes into a DNS server to match a domain name with the wrong IP address, allowing the attacker to send users to a fake website and exposing them to even more harm. DNS and IP are designed to scale now matter how much internet grows.
