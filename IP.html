IP Addresses, Packets, and Routing

What is an IP Address?

IP stands for Internet Protocol. The internet is actually a design philosophy and an architecture expressed in a set of protocols.

A protocol is a well known set of rules and standards used to communicate across machines. In order for a new technology to use the internet, it just needs to know which protocols to work with.

Every device has a different addresses, which is a set of numbers unique to each device on the network. The addressing system for computers on the internet forms the internet protocol or IP.

*** insert ip address description picture**

An IP address is a group of numbers, organized in a hierarchy. Each number is made up of bits; there are normally 32 bits or 8 bits per each part of the address.

The numbers are broken down by networks, subnetworks, and finally the specific address. This naming convention is called IPv4 and it provides for more than 4 billion unique addresses. With the tremendous growth in both internet users and devices, IPv4 will not have enough unique addresses to last. We are transitioning to a longer IP address format called IPv6, which will use 128 bits per address. This provides 340 undecillion unique addresses, or more than enough for every grain of sand on earth to have its own address.

Visiting a website is really just your computer asking another computer for information. Your computer sends a message to the other computer's IP address and it also sends its origin address so the other computer knows where to send information to.

What is ICANN?
  • ICANN is the Internet Corporation for Assigned Names and Numbers. It is a non-profit organization that:
    • Coordinates maintenance and procedures of several databases related to the namespaces and numerical spaces of the internet
    • Ensures the network 's stable and secure operation
    • Performs the actual technical maintenance work of the Central Internet Address pools and DNS root zone registries pursuant to the Internet Assigned Numbers Authority (IANA) function contract

Simply put, ICANN maintains and identifies IP addresses to provide a global, scalable internet. It has a multi-stakeholder model that allows for vast representation of those in the global internet community. ICANN promotes competition and innovation in the DNS marketplace, and it develops policies using many checks and balances to ensure the best environment for the global internet community.

Prior to ICANN, the function of administering registries of IP addresses was performed by Jon Postel, a CS researcher involved in the creation of ARPANET. Postel no longer wanted to be involved, so the government ICANN to manage the IANA.
