W3C and HTML and CSS

What is W3C?

W3C stands for World Wide Web Consortium. It is the main international standards organization for the World Wide Web. Founded in 1994, and lead by Tim Berners Lee, the organization develops protocols and guidelines that ensure long term web growth.

W3C sets standards to define key parts of what makes the World Wide Web work. The World Wide Web(W3) was created in 1989 by Tim Berners Lee. He wrote the first Web Browser, server, and home page and wrote the first specifications for URLs, HTTP, and HTML. W3C defines the World Wide Web as an information space in which the items of interest, referred to as resources, are identified by global identifiers called Uniform Resource Identifiers (URI)." W3C publishes documents that define Web technologies. These documents follow a process designed by W3C to promote consensus, fairness, public accountability, and quality. At the end of this process, W3C publishes recommendations, which are considered web standards. Member organizations, a full time staff, and the public work to develop these web standards. This is done to ensure that the positive benefits of the web are available to all people, like communication and commerce.

What is HTML?

HTML stands for Hypertext Markup Language. It is the standard markup language that defines the structure of a web page. HTML consists of a series of elements that tell the browser how to display the content. It can be thought of as a really big simple document that formats the page. So, HTML is used to describe and determine the content of a web page (which lines are headers vs paragraphs) and also includes the web page's text. It does not contain images and videos, though, as they are included in separate files.

What is CSS?

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on the screen. It is used define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes. CSS saves a lot of time and work, you can change the look of an entire website with just one file, whereas if you were using HTML, you would have to define the fonts and colors on every page.
